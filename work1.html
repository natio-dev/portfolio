Works — 01

結婚祝いWebサイト

  • HTML
  • CSS
  • JavaScript
  • Supabase
  • Animation

Live Demo

wedding-demo.html

概要

友人の結婚をお祝いするために個人で制作したお祝いメッセージサイトです。当日までサプライズとして準備し、式の前日に二人へ共有しました。招待客がスマホからメッセージを投稿・閲覧できる仕組みを実装し、会場でも使ってもらえる設計にしました。

制作期間

約2週間

制作形態

個人制作

公開状況

限定公開（招待制）

使用技術

HTML / CSS / JS / Supabase

工夫した点

01

Supabaseでリアルタイム投稿を実現

メッセージの投稿・取得にSupabaseのデータベースを活用。ページをリロードしなくても新しいメッセージがリアルタイムで表示されるよう実装しました。

02

アニメーションで感動を演出

CSS AnimationとJavaScriptを組み合わせ、花びらが舞うエフェクトや、メッセージカードがふわっと現れる演出を実装。「見た瞬間に泣いた」という声をもらえました。

03

スマホで操作しやすいUI設計

式の当日にゲストがスマホから投稿することを想定し、入力フォームのタップ領域や文字サイズをHCDの観点から設計。操作で迷わせない導線を意識しました。
